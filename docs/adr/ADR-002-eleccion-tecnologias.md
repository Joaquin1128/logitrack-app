# ADR 002 – Elección de tecnologías

## Contexto
El proyecto requiere tecnologías conocidas por el equipo, con buena documentación
y adecuadas para desarrollar un MVP académico con foco en gestión y trazabilidad.

## Decisión
Se adopta el siguiente stack tecnológico:
- Backend: Java con Spring Boot
- Frontend: React
- Comunicación: API REST
- Persistencia: Base de datos relacional

## Consecuencias
- Amplio soporte y comunidad
- Separación clara entre frontend y backend
- Mayor claridad en la arquitectura del sistema
- Mayor complejidad inicial que soluciones monolíticas simples
